**Инструкция по настройке:**

**Получите API ключ VirusTotal:**
  * Зарегистрируйтесь на virustotal.com
  * Перейдите в настройки аккаунта
  * Скопируйте API ключ
  * Замените "ВАШ_API_КЛЮЧ" на ваш ключ

**Подготовьте логи Suricata:**
  * Убедитесь, что Suricata настроена и генерирует логи в формате JSON
  * Укажите правильный путь в SURICATA_LOG_PATH
  * Или создайте тестовый файл с примером логов

**Настройте подозрительные IP:**
  * Добавьте IP-адреса для проверки в список SUSPICIOUS_IPS

--------------------------------------------------------------------------
**Особенности реализации:**
**Два источника данных:**
- VirusTotal API для проверки репутации IP-адресов
- Логи Suricata

**Обнаружение угроз:**
- Подозрительные IP-адреса
- DNS-флуд (частые DNS-запросы)
- Алерты Suricata
- Критические уязвимости (имитация)

**Реагирование:**
- Блокировка подозрительных IP
- Применение патчей для уязвимостей
- Изоляция зараженных систем
- Уведомления администратору

**Отчетность:**
- JSON отчет с деталями угроз
- CSV файл для дальнейшего анализа
- Визуализация с 4 графиками
